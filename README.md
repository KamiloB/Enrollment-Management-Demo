# Enrollment Demo

🌎 English version below | 🇪🇸 Versión en español más abajo

---

🌎 English
> Extracted from a real-world SaaS system.

### What this is

A standalone enrollment workflow demo with a public-style multi-step form and an internal review board.

---

### Problem it solves

Enrollment teams need to capture complete student records while reducing incomplete submissions and improving review speed.

---

### What you can do in this demo

- Complete a step-by-step enrollment form
- Validate data at each stage before continuing
- Generate `pending` applications
- Review submissions in an internal panel
- Approve or reject records

---

### What it includes

- Multi-step form flow with per-step validation.
- Structured form state for training, student, health, guardians, and consent data.
- Realistic status lifecycle: `pending`, `approved`, `rejected`.
- Internal moderation actions from the same UI.
- Utility-driven validation logic in `src/utils/validators.js`.

---

### How to run

- npm install
- npm run dev
